 Employee Management System using Dapper

 Overview
A simple console application built with .NET and Dapper to manage employees and departments.

 Features
- Basic CRUD operations (Insert, Update, Delete, Get)
- Retrieve employee by Id
- Scalar queries (count, max salary, average salary)
- Multi-mapping between employees and departments
- Using stored procedures for insert and retrieval

 Technologies
- C#
- .NET
- Dapper
- SQL Server
